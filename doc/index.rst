ALPS-CCD: A toolkit to work with CCD data
=========================================

The ALPS-CCD package contains tools to open CCD frames saved in binary
formats (currently FITS, SPE and also greyscale PNG). In addition, it
contains tools to normalize and analyze CCD frames:

* :mod:`ccd.io` for opening and storing data.
* :mod:`ccd.analysis` for analysis tools.
